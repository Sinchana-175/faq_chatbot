# Smart FAQ Chatbot

This is a Smart FAQ Chatbot that answers syllabus-related questions using a local LLM (Llama 3 via Ollama).

## Features
- Multi-subject support (DS, DBMS, OS, ML)
- Context-aware responses
- Prompt-based LLM interaction
- Streamlit frontend

## Technologies Used
- Python
- Ollama
- Llama 3
- Streamlit

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Start Ollama:
ollama run llama3

3. Run app:
streamlit run streamlit_app.py

## Example Questions
- What topics are covered in ML?
- Explain stack
- What is regression?
